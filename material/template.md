# Minimum Presentation Template

## Slide 1 – Title Slide

* Project title
* Group number
* Names of team members
* Course name

---

## Slide 2 – Project Overview

* What problem does the project solve?
* Why is the project useful or interesting?
* Target users or use case

Example:

* AI assistant for healthcare advice
* Security incident classifier
* Business chatbot

---

## Slide 3 – Project Architecture

Include a simple diagram showing:

* User input
* Gradio interface
* AI model/API
* Output generation

Optional:

* Data flow between components

---

## Slide 4 – Phase 1: UI Prototype

Show:

* The original Gradio prototype
* Main interface features
* Mock/demo data used

Explain briefly:

* Design decisions
* User interaction flow

---

## Slide 5 – Phase 2: Gemini API Integration

Show:

* How Gemini was connected
* Prompting approach
* Example outputs

Discuss briefly:

* What worked well
* Main challenges

---

## Slide 6 – Phase 3: ML Model / Fine-Tuned Model

Show:

* The additional ML model used

  * Fine-tuned LLM OR classical ML model
* How it was integrated
* Example outputs

Briefly explain:

* Training or setup process
* Any limitations

---

## Slide 7 – Comparison: Gemini vs Phase 3 Model

Compare:

* Output quality
* Speed
* Accuracy/relevance
* Cost/resources
* Strengths and weaknesses

You may use:

* Small table
* Example comparison screenshots

---

## Slide 8 – Live Demo

Demonstrate:

1. Phase 1 prototype
2. Phase 2 working system
3. Phase 3 working system
4. Comparison between Phase 2 and Phase 3 outputs

---

## Slide 9 – Challenges and Lessons Learned

Discuss:

* Technical challenges
* Team collaboration
* What you learned

---

## Slide 10 – "The 4Ls" Reflection

Briefly discuss:

* **Loved:** What worked well?
* **Loathed:** What challenges or problems did you face?
* **Longed for:** What would have helped improve the project?
* **Learned:** What new skills or knowledge did you gain?


---

## Slide 11 – Team Contributions

Briefly describe:

* What each team member contributed

Example:

* UI development
* API integration
* ML implementation
* Testing/documentation

---

## Slide 12 – Conclusion

* Summary of the project
* Future improvements
* Final thoughts
